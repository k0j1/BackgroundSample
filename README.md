背景パターン

現時点で作成している背景は７つあります。

　・水玉模様(隙間少なめ、グラデーションあり)
　・チェック柄(モノクロ)
　・多数の丸(カラフル)
　・多数の四角(カラフル)
　・フラットデザイン文字入り(カラフル、半透明)
　・フラットデザイン文字入り(カラフル、透過なし)
　・フラットデザインの市松模様(カラフル)
